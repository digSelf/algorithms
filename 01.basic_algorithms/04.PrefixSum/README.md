# 01.一维数组的前缀和

输入一个长度为 n 的整数序列。

接下来再输入 m 个询问，每个询问输入一对 l,r。

对于每个询问，输出原序列中从第 l 个数到第 r 个数的和。

## 输入格式

第一行包含两个整数 n 和 m。

第二行包含 n 个整数，表示整数数列。

接下来 m 行，每行包含两个整数 l 和 r，表示一个询问的区间范围。

## 输出格式

共 m 行，每行输出一个询问的结果。

## 数据范围

* `1 <= l <= r <= n`
* `1 <= n,m <= 100000`
* `−1000 <= 数列中元素的值 <= 1000`

## 输入样例：

```
5 3
2 1 3 6 4
1 2
1 3
2 4
```

## 输出样例：

```
3
6
10
```

# 02. 二维矩阵的前缀和

输入一个 n 行 m 列的整数矩阵，再输入 q 个询问，每个询问包含四个整数 x1,y1,x2,y 表示一个子矩阵的左上角坐标和右下角坐标。

对于每个询问输出子矩阵中所有数的和。

## 输入格式

第一行包含三个整数 n，m，q

接下来 n 行，每行包含 m 个整数，表示整数矩阵。

接下来 q 行，每行包含四个整数 x1,y1,x2,y2，表示一组询问。

## 输出格式

共 q 行，每行输出一个询问的结果。

## 数据范围

* `1 <= n,m <= 1000`
* `1 <= q <= 200000`
* `1 <= x1 <= x2 <= n`
* `1 <= y1 <= y2 <= m`
* `−1000 <= 矩阵内元素的值 <= 1000`

## 输入样例：

```
3 4 3
1 7 2 4
3 6 2 8
2 1 2 3
1 1 2 2
2 1 3 4
1 3 3 4
```

## 输出样例：

```
17
27
21
```
